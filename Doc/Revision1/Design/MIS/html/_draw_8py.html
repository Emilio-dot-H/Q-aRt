Detailed Description

Draw

Date
9/11/2016 This class generates a PNG image file representing the data contained in the QR code matrix.

This class takes the matrix generated by the matrix module and creates a PNG formatted image of the QR code to be created by the program.
State variables: none

image = Draw.drawQRCode(absPath, qrmatrix)

Method to combine QR code with another image.

Author

Method accepts eight parameters.

Parameters
verInteger between 1 and 40 that specifies the desired size of the QR code.
qr_nameAccepts a string indicating the name of the saved QR code image
bg_nameAccepts a string indicating the name of the saved background image to be combined
colorizedAccepts a boolean indicating whether the result should be in black and white(False) or in colour (True)
contrastAccepts a double representing the relative contrast of the output image. A value of 1.0 makes no change.
brightnessAccepts a double representing the relative brightness of the output image. A value of 1.0 makes no change.
save_dirAccepts a string indicating the name of the directory to saved the cominbed image in.
save_nameAccepts a string indicating the save name of the output comined image.
Returns
Saves a PNG, JPG, or GIF file to the save_dir directory
