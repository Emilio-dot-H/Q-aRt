Detailed Description

QR Code Module

Date
4/11/2016 This class places the data into a QR code.

This class creates a matrix, places structures required according to QR code standards, evaluates the best mask pattern and applies it to the data.
State variables:
int array qrmatrix - 2D array representing the QR code matrix.

QRMatrix = Matrix.getMatrix(2, 'L', finalData)
