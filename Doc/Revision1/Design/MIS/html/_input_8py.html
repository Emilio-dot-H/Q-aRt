Method to create QR code using command prompt.

Date
4/11/2016

Method accepts nine parameters.

Parameters
versionInteger between 1 and 40 that specifies the size of the QR code.
eclAccepts one of four values: 'L', 'M', 'Q', 'H' that specify the level of error correction to be encoded.
pictureAccepts the file location of an image to combine
colorizedAccepts a boolean indicating whether or not the output will be in colour
contrastAccepts an integer indicating the cotrast level
brightnessAccepts an integer indicating the brightness level
saveNameAccepts a string containing the desired save name for the finished qr code
saveDirectoryAccpets a string contating the desired save location for the finished qr code
Returns
Final binary string representing the data.
