Detailed Description

Constant

Date
13/11/2016 This class contains useful constants for qr processing.

Module is taken from original project. Contains useful constants for QR code computation.
State variables:
char_cap - list of character capacities for each version and error correction level
required_bytes - list of required bytes for each version
num_list - list of valid characters for numeric mode
alphanum_list - list of valid characters for alphanumeric mode
grouping_list - list of codeword grouping into blocks for each version
mode_indicator - list of encoding mode strings
GP_list - lsit of generator polynomials
ecc_num_per_block - list of error correction codewords per block for each version and error correction level
po2 - list of powers of two in Galois Field
log - list of logarithms in Galois Field
required_remainder_bits - list of required remainder bits for each version for interleaving
alig_location - list of alignment pattern locations for each version
format_info_string - list of format strings for each mask pattern
version_info_string - list of version strings for each version
