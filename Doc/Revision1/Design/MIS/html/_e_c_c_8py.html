Detailed Description

ECC

Date
2/11/2016 This class generates error correction codewords.

This class generates error correction codewords for a given version, error correction level and data string.
State variables:
int list MP - coefficients representing the message polynomial at each iteration.
int list GP - coefficients representing the generator polynomial at each iteration.

codewords = ECC.getCodewords(1, 'Q', data)
