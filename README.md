# Q-aRt Code

Team Name: Q-aRt QTs

Team Members: Elton Schiott, Emilio Hajj, Liam Duncan

This project is a reimplementation of an artistic QR encoder located at https://github.com/sylnsfar/qrcode

Dependencies:

Python 3.4
Pillow 3.3.1
numpy 1.12
imageio 1.6

Usage:

Via command prompt:

python Main.py -gui
